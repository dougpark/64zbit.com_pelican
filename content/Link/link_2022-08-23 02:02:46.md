> New installation¶
> 
> automatic (recommended)¶
> 
> Install curl:
> 
> 
> $ sudo apt install -y curl
> Run the following command:
> 
> 
> $ curl -fsSL https://raw.githubusercontent.com/SensorsIot/IOTstack/master/install.sh | bash
> Run the menu and choose your containers:
> 
> 
> $ cd ~/IOTstack
> $ ./menu.sh
> Bring up your stack:
> 
> 
> $ cd ~/IOTstack
> $ docker-compose up -d
>
